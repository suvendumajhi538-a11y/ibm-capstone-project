# User Story: Item Listing
**As a** registered user,
**I want to** list my household items,
**So that** others can see and recycle them.

## Details and Assumptions
- User must be logged in.
- Item must have a title and category.

## Acceptance Criteria
1. Given I am on the listing page, when I enter valid details, then the item should be saved.
2. Given I leave the title blank, when I try to save, then an error should appear.
